

1. Copy the folder on to your Desktop.

2. Open Terminal.

3. type 
		cd ./Desktop/Monster_Job_Search/
		chmod +x ./Setup.sh
		./Setup.sh

4. An installation will start. (Enter your password when ever prompted)

5. At one point the installation will pause,
	type export PATH=/usr/local/bin:$PATH

6. Press control + x

7. Press y

8. Press Enter

9. After installation, type,
		python3 Monster_Main.py

10. Enter inputs.

11. A .JSON file is created in the folder.
